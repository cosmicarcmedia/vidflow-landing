Bulk actions

Updated July 10, 2026

Bulk actions change many library items in one pass instead of opening them one at a time. They run from the Bulk Manage Content dialog.

Note

Note: Bulk actions are available on plans that include them. See Plans and features.

#How it works

Every bulk action is the same three-step shape: pick an action tile, select the items it should touch, then apply. The action runs as one batch request, so you get a single confirmation with a count rather than one confirmation per item. Bulk Actions in the sidebar opens the library straight into this dialog.

#Open Bulk Manage

  1. In the rail, click Content Library, then Bulk Actions.
  2. The library opens with the Bulk Manage Content dialog and four actions:
    • Re-Tag Content: add or override tags on the selected items.
    • Select Default Accounts: add or override the default accounts the selected items post to.
    • Delete: delete multiple items at once.
    • Copy Content to Other Workspaces: copy the selected items into another workspace.

#Select the content

Every action starts with the same picker. Click Select content to manage, then check the items to include. Use the search box to narrow the list, and Select All or Deselect All to move faster. The dialog shows how many items are selected.

#Re-tag content

  1. Choose Re-Tag Content and select your items.
  2. Pick the tags to apply. You can create a new tag inline.
  3. Check Remove Existing Content Tags to replace the items' tags instead of adding to them.
  4. Apply. A confirmation reports how many items were re-tagged.

See Tags for how tags drive filters, schedules, and automation.

#Set default accounts

  1. Choose Select Default Accounts and select your items.
  2. Pick the connected accounts the items should post to. Select All Accounts picks everything.
  3. Check Remove Existing Default Accounts to replace instead of add.
  4. Apply.

#Copy content to another workspace

Useful when you run more than one brand and want to reuse content.

  1. Choose Copy Content to Other Workspaces and select your items.
  2. Pick the Destination Workspace. If the workspace switcher has nothing else to offer, the dialog says "No other workspaces available."
  3. Apply. The items are copied; the originals stay in the current workspace.

See Running multiple brands.

#Delete in bulk

  1. Choose Delete and select your items.
  2. Confirm. The dialog warns that this permanently deletes the selected items and cannot be undone.
Important

Important: Bulk delete is permanent. There is no undo and no trash.

#Adding content in bulk

Bulk Manage edits content that already exists. To add many items at once, including CSV import, use the Bulk Upload page instead. See Bulk upload and CSV import.

#Get better results

  • Use the search box inside the item picker before selecting, especially in a large library. Narrowing the list first makes it much easier to confirm you selected the right items before you apply an action that cannot be undone.
  • Re-tag in two passes for a clean take-over: apply the new tag first, then re-open Bulk Manage with Remove Existing Content Tags checked to strip only the old one. Doing both in one pass removes tags before the new ones are confirmed to be right.
  • Set default accounts in bulk right after a CSV import or a large upload, so every new item has somewhere to publish instead of needing that set one item at a time.
  • Before a bulk delete, sort or filter the library down to exactly the set you intend to remove and eyeball the count in the dialog. There is no trash to recover from afterward.

#Troubleshooting

  • The action tiles are greyed out or clicking New Flow-style upgrade prompts appear. Bulk actions require a plan that includes them. See Plans and features.
  • Copy to Other Workspaces says no other workspaces are available. You need at least one other workspace to copy into. See Running multiple brands.
  • A re-tag or default-accounts apply did not change anything. Confirm items were actually checked in the picker; the dialog's selected count at the top is the fastest way to verify before applying.
  • I need to bulk-add content, not edit it. Bulk Manage only acts on existing items. Use Bulk upload and CSV import to add many items at once.
