Edit a post

Updated July 9, 2026

Every item in the Content Library opens in the same editor. This article walks through editing a text post or image, then summarizes how clips and episodes differ.

#Open the editor

  1. Open the Content Library and click any item. It opens read-only first.
  2. Click Edit Content to make changes.
  3. Use the arrows at the top to step to the previous or next item. A counter shows your position in the current list.
  4. Click Save when you are done, or Cancel to discard changes.

#What you can edit

  • Internal Title: a name for your own reference. It is optional and never published.
  • Caption (image posts) or Post Content (text posts): the text that publishes. A character counter tracks the limit for each platform.
  • Tags: click tag pills to add or remove them. See Tags.
  • Post to: toggle the platforms this item publishes to.
  • Status: Draft, Queued, Scheduled, or Published. See Content Library basics for what each means.

On image posts, Image Studio opens an image panel beside the editor. See AI images and thumbnails.

Note

Note: When TikTok is toggled on under Post to, a TikTok settings panel appears for audience, comments, and content disclosure. See Post now for the details.

#Per-platform captions

One item can carry a different caption for each platform.

  1. Turn on the Platform specific captions switch.
  2. A tab appears for each platform selected under Post to.
  3. Write a caption in each tab. Leave a tab blank to use the main post content.
  4. Each tab has its own character counter against that platform's limit.
  5. Generate all platforms drafts a caption for every platform tab at once.
Note

Note: Per-platform captions are available on plans that include them. See Plans and features.

#Carousel images

Image posts can hold up to 10 images as a carousel. The Carousel Images section shows the count, and you can Add current image or Add Images to drop files or browse.

Note

Note: Carousels are available on plans that include them. See Plans and features.

#Approval and publishing actions

The editor footer holds the item-level actions:

  • Submit for Approval appears while the item's approval state is draft. After you submit, a Pending Approval badge shows until a reviewer acts, then the badge becomes Approved or Rejected. See Approval status in your library.
  • Post Now publishes the item immediately to a connected platform. See Post now.
  • Delete permanently removes the post after a confirmation. This cannot be undone.
  • Close leaves the editor.

#Editing clips

Clips open in this same editor with extra tools: the clip transcript, AI Clip Writing, thumbnail options, and Replace content. See Edit clips: captions, reframing, thumbnails.

#Editing episodes

Episodes open in the same modal as Edit Episode, with four tabs:

  • Details: thumbnail, title, description, episode and season numbers, tags, the media file, and the episode status (draft, processing, ready, published, archived).
  • Show Notes: supports Markdown and appears in podcast players and YouTube descriptions.
  • Transcript: pull or re-pull the transcript and download it as a .txt file. See Transcripts.
  • Generated Clips: clips created from this episode.

Save episode changes with Save Episode.

Note

Note: Episodes are deleted from the Episodes page, not from this editor.
