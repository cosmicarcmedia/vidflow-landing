Import from YouTube

Updated July 10, 2026

Bring videos into Vidflow as episodes straight from YouTube, either by browsing your own connected channel or by pasting links to any videos you want. Either way, each import pulls the video's metadata and transcript only. The video file itself is never downloaded.

#How it works

Both import modes create one episode per video and start pulling transcripts in the background, so you can keep working while they process. Neither mode downloads the actual video file, only its metadata and transcript, so a YouTube-imported episode has no attached media. That single fact drives one real limitation: native clip generation needs an uploaded video file to cut from, and a YouTube import never provides one. If you want clips from a video you import, upload the file separately for that episode.

#Import from your connected channel

Use this mode when the videos you want are already on your own YouTube channel.

  1. On the Episodes page, click Import Episodes. You can also click Make Content, then Import Episodes, from anywhere in the app.
  2. In the import dialog, make sure Connect channel is selected at the top.
  3. Choose the Videos or Shorts tab. The dialog lists your channel's uploads. Click Load more to page further back.
  4. Select the videos you want. You can import up to 50 videos at a time. Videos already in your workspace show an Imported badge and cannot be selected again.
  5. Click Import into Library.

This mode requires a connected YouTube channel. Connect it in Brand Settings > Connected Accounts first; if it isn't connected, the dialog prompts you to connect it or switch to Paste links instead. See Connect YouTube.

#Import by pasting links

Use this mode for any YouTube video, whether or not it's on a channel you've connected, someone else's video, a link a client sent you, anything with a public YouTube URL.

  1. Open the same Import Episodes dialog and click Paste links at the top.
  2. Paste one or more YouTube URLs into the text box, separated by spaces or new lines.
  3. Click Import into Library. The button shows how many links you've pasted.

Vidflow fetches each video's title automatically and creates one transcript-only episode per link. Duplicates already in your workspace are skipped and reported, invalid links (not a recognizable YouTube URL) are skipped and reported separately, and imports that would exceed your plan's monthly episode allowance are skipped with a note to upgrade in Settings, Billing. Successfully imported episodes start pulling transcripts right away.

#What gets imported

  • The video's metadata (and, for a connected-channel import, its title, view count, thumbnail, and duration shown in the picker) and its transcript.
  • A title fetched automatically for pasted links.
  • Not the video file. The episode links back to the original video on YouTube.
Note

Note: Imported episodes count toward your plan's monthly episode allowance, whichever mode you use. If an import would go past it, the extra videos are skipped and Vidflow tells you how many. See Usage limits and monthly allowances.

#Get better results

  • If you want clips from a video you're importing, upload the file for that episode too, either mode alone leaves you with posts, images, and captions, but no clips.
  • Use Paste links for anything outside your own channel, guest appearances, a client's video, competitor research, it works on any public YouTube URL.
  • Paste several links at once instead of importing one at a time, both modes accept a batch and report duplicates, quota skips, and failures together so you only have to check one summary.
  • If a transcript comes back thin or missing, the source video may lack usable captions. Upload the actual file instead so Vidflow can transcribe the audio directly rather than relying on YouTube's captions.

#Troubleshooting

"Connect your YouTube channel." The connected-channel mode has no YouTube connection. Connect it in Brand Settings > Connected Accounts, or switch to Paste links in the same dialog, which needs no connection at all.

Videos skipped with "monthly episode limit reached." The import hit your plan's episode allowance partway through. The rest import once your allowance resets, or after an upgrade from the Billing page in the app.

Some pasted links were skipped as invalid. The link wasn't a recognizable YouTube URL. Double check it's a full youtube.com or youtu.be link, not a shortened or malformed one.

"YouTube's daily API quota has been reached. Please try again later." YouTube limits how often Vidflow can read channel or video data each day. Wait and run the import again later.

Transcript missing after import. Some videos take longer, and a few have no usable captions. Open the episode's Transcript tab and use Pull transcript to retry. See Transcripts.
