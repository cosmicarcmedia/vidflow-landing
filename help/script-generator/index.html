Script Generator

Updated July 9, 2026

The Script Generator writes a complete script for a YouTube video, podcast episode, or short-form video in your brand voice, sized to the runtime you want. You edit it as one document, refine any passage with AI, and save it to your script library.

Find it under Magic Tools in the left rail. Script Library, beneath it, lists everything you have saved.

#Generate a script

  1. Open Script Generator and click New Script.
  2. In Generation settings, pick a format: YouTube Video, Podcast Episode, or Short-Form.
  3. Set Target Video Length in minutes, from 1 to 180 (10 is the default). The panel estimates the spoken word count at the format's speaking pace: 145 words per minute for YouTube, 150 for podcast, 160 for short-form.
  4. Describe the topic in What's this script about?
  5. Click Generate Script.

Note: Scripts read like you only after your brand voice is set up; until then the page shows a banner and the AI writes in a generic voice. See Set up your brand voice. Each generation counts toward your plan's monthly AI allowance. See Usage limits and monthly allowances.

#Edit the script

The script opens in a single-document editor with a live word count and an estimated runtime.

  • Select any passage and right-click to Rewrite, Expand, or Shorten it. The updated text replaces your selection in place.
  • Pull in saved ideas from the Idea Library panel on the right: drag an idea into the script, or click it to insert it.
  • Regenerate Script rewrites the whole script from your current settings.

#Save and manage scripts

  • Click Save Script to add the script to your library, or Update Script to save changes to an existing one.
  • Script Library lists your saved scripts with a Final or Draft status.
  • Deleting a script asks you to confirm first. Deletion cannot be undone.
  • Resetting the editor clears the current script and settings after a confirmation. Unsaved changes are lost, so save first.

#Use a script elsewhere

Saved scripts are a source for other Vidflow tools. Generate a batch of text posts from a script in Magic Text Posts, or turn one into a long-form article with the Blog Post Generator.
