Auto-import from Google Drive or Dropbox

Updated July 10, 2026

Point Vidflow at a folder in Google Drive or Dropbox and every new file becomes a clip in your Content Library, transcribed, AI-titled, and captioned in your brand voice, with your tags already applied.

#How it works

  • Each source watches one cloud folder. New files are imported hourly, or on demand, as clips in the Content Library.
  • Each imported clip is transcribed, AI-titled, and captioned. Your brand voice is always included in the AI instructions.
  • Tags you choose are applied automatically, so tag-fed schedules pick the content up on their own. See Schedules and queues.
  • Files are never deleted from the source folder, and files already imported are skipped on later polls.
  • By default a source only imports files added after you create it. Use Import existing to backfill older files.

#Create a source

  1. In the navigation rail, open Content Library, then Auto-Import.
  2. Click New Source.
  3. Choose Google Drive or Dropbox and connect your account. Vidflow asks for read-only access.
  4. Pick the folder to watch in the Select Folder to Watch dialog.

New files in that folder now import automatically every hour.

#Source settings

Open a source's settings to control what happens to each imported clip:

  • Auto-Apply Tags: tags added to every imported clip. Create tags on the Tags page first if you have none. See Tags.
  • Require Approval: imported clips are marked "Needs Approval" and will not publish until manually approved. See Approval status in your library.
  • Platform-Specific Captions: pick target platforms (Instagram, TikTok, Facebook, LinkedIn, YouTube Shorts, X (Twitter), Threads) and the AI writes a caption tuned to each one.
  • Title Generation Prompt and Caption Generation Prompt: replace the default AI instructions with your own. The Magic prompt button drafts one for you.

#Run and monitor a source

  • Import Now polls the folder on demand instead of waiting for the hourly run.
  • Import existing imports files that were already in the folder before you connected it.
  • The import log lists every file the source has seen with its result: imported, failed, or skipped.
  • You can change the watched folder at any time from the source card.
  • Deleting a source stops monitoring the folder. Already-imported content is not affected.

#Troubleshooting

New files are not showing up. Sources poll hourly, so a fresh file can take up to an hour to appear. Click Import Now to poll immediately. If the file was in the folder before you created the source, run Import existing instead. The import log shows anything that failed or was skipped.

Clips arrive without tags. Set Auto-Apply Tags in the source settings. Tags must exist on the Tags page before you can select them.

#Get better results

  • Turn on Require Approval for any source a client or teammate drops footage into, so nothing publishes until someone reviews it first.
  • Write a custom Title Generation Prompt and Caption Generation Prompt if the defaults don't sound like your brand, or click Magic prompt to have the AI draft a starting point you can edit.
  • Set Auto-Apply Tags before the first import runs, not after, so tag-driven schedules pick content up from day one instead of catching up later.
  • Use Import existing right after creating a source if the folder already has content in it, otherwise those older files sit untouched until you run it manually.
