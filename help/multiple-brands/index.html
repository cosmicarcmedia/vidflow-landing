Running multiple brands

Updated July 9, 2026

Agencies running content for more than one client use a separate workspace for each brand.

#One workspace per brand

Each workspace keeps its content, connected accounts, brand voice, and approval links completely separate from every other workspace. Nothing you set up for one client brand shows up in another, so it is safe to run several client brands from the same Vidflow account. See Workspaces and brand spaces for what a workspace isolates.

#Adding more brand spaces

Every account includes one workspace on any plan. Running more than one brand requires the Agency plan. Beyond the workspace your plan includes, Agency owners can purchase additional brand spaces as an add-on, added only as you need them.

Current pricing for the base plan and for additional brand spaces lives on the Billing page in the app, or on vidflow.io.

Note

Note: If you try to create a workspace beyond what your plan allows, Vidflow shows a clear in-app message telling you what to do next, either upgrade or add a brand space.

#Copying content between workspaces

If you have already built out one client's content and want to reuse pieces of it for another brand, you do not need to rebuild from scratch. Bulk Manage includes a Copy Content to Other Workspaces action that copies selected library items into a different workspace, leaving the originals in place. See Bulk actions for the full steps.

This is available on plans that include bulk actions. See Plans and features.
